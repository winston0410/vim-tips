# vim-tips

## Tricks

### Search and replace

- `/` + `cgn`

### Change current word under cursor

- `cw` or `ciw`

### Change words inside quote, double quote or    

- `ci"` to change everything inside `"`
- `ci'` to change everything inside `'`
- `ci(` to change everything inside `(`

https://stackoverflow.com/questions/11630440/how-to-replace-text-between-quotes-in-vi
